1. Data Collection & Source

Cut Nerd Solutions follows the principle of data minimization. We only collect essential information authorized via official API protocols (OAuth 2.0). This may include:

  • Basic profile identifiers (e.g., public display name, avatar).
  • Technical scopes required for video management and publishing.
  • Temporary access tokens necessary for automated workflow execution.

2. Purpose of Processing

The collected data is processed exclusively for workflow automation and media synchronization. We do not engage in user behavior profiling, third-party tracking, or data monetization. All processing is strictly limited to the functional requirements of the integration.

3. No Third-Party Disclosure

We maintain a strict Zero-Sharing Policy. Your data is never sold, leased, or distributed to third-party marketing entities. Access is limited only to the technical environment required to communicate with the destination platform's API.

4. Security and Encryption

All API keys and OAuth tokens are stored using industry-standard encryption within a private server environment. We implement rigorous security measures to prevent unauthorized access, maintain data accuracy, and ensure the appropriate use of information.

5. Data Retention & Deletion

Users maintain full control over their data. Access can be revoked at any time through the security settings of the connected social network. Upon revocation, all associated tokens are rendered invalid immediately and purged from our execution logs within 24 hours.
