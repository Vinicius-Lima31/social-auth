1. Coleta de Informações

O aplicativo pessoal Cut Nerd coleta apenas as informações estritamente necessárias fornecidas através da API oficial do TikTok via protocolo de autorização OAuth 2.0. Isso inclui:

  • Informações básicas de perfil (username, avatar).
  • Permissões de upload e gerenciamento de vídeos.
  • Tokens de acesso temporários necessários para a execução dos fluxos.

2. Uso dos Dados e Finalidade

A finalidade exclusiva do processamento é a automação de workflows de publicação via n8n para o canal @cut.nerd. Não realizamos análise de comportamento, rastreamento de terceiros ou qualquer forma de marketing invasivo com os dados coletados.

3. Armazenamento e Segurança

Todos os tokens e chaves de API são armazenados de forma criptografada em ambiente de servidor privado controlado pelo usuário. Nenhuma informação é transmitida para bancos de dados públicos ou terceiros não autorizados.

4. Exclusão e Retenção

O usuário pode revogar o acesso do Cut Nerd a qualquer momento através das configurações de segurança do TikTok. Ao revogar o acesso, os tokens tornam-se inválidos imediatamente e são eliminados dos logs de execução do sistema em até 24 horas.
